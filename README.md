# 🛡️ Security Equipment Management System

A Full Stack Web Application developed during my **HPCL Internship** for managing security equipment inventory. The application enables users to efficiently add, update, search, monitor, and manage equipment through an interactive dashboard.

---

## 📖 Project Overview

The Security Equipment Management System is designed to simplify the management of security-related equipment used in industrial environments. It provides an intuitive interface for maintaining equipment records and visualizing equipment statistics through charts and dashboard summaries.

---

## 🌟 Highlights

- Full Stack Web Application
- RESTful API using Spring Boot
- Interactive React Dashboard
- Equipment Analytics with Charts
- CRUD Operations
- Search & Pagination
- Responsive Bootstrap UI
- SweetAlert2 Notifications
- MySQL Database Integration

## 🚀 Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/Srivalli0612/security-equipment-management-system.git
```

---

### 2. Backend Setup

```bash
cd backend
```

Configure your MySQL database in:

```
src/main/resources/application.properties
```

Run the Spring Boot application:

```bash
./mvnw spring-boot:run
```

Backend runs on:

```
http://localhost:8081
```

---

### 3. Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

Frontend runs on:

```
http://localhost:5173
```
